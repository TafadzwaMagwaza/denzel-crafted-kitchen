**Denzel Crafted Kitchen** is a sophisticated digital portfolio designed to showcase the culinary expertise of Chef Denzel. It serves as the primary touchpoint for clients seeking private dining experiences, catering services, and gastronomic consultation.

The website reflects the chef's philosophy: food is art. With a clean, minimalist design, the application allows the vibrant photography of the dishes to take center stage, while providing a seamless booking experience for potential clients.

## ğŸš€ Getting Started

To serve this project locally on your machine:

### Prerequisites

*   Node.js (v16+)
*   npm or yarn

### Installation

1.  **Clone the repository**
    ```bash
    git clone https://github.com/Dreamvillians/denzel-crafted-kitchen.git
    ```

2.  **Enter the kitchen**
    ```bash
    cd denzel-crafted-kitchen
    ```

3.  **Mise en place (Install dependencies)**
    ```bash
    npm install
    ```

4.  **Start cooking**
    ```bash
    npm run dev
    ```

5.  Open `http://localhost:5173` to taste the app.
